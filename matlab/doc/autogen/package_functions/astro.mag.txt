Package: astro.mag

magVega - Return the Vega magnitude adopted by GAIA (0.023\pm0.008) Example: Mag=astro.mag.magVega

photonIntegral - Calculate the number of photons from a spectrum given transmission.

specVega - Return the Vega (Alpha Lyr) standar spectrum for Vega mag calculations

synMag - Calculate the synthetic magnitude of a spectrum and transmission in the Vega, AB, or STMAG systems.
